###  [Python 程序设计语言](#)

**介绍:** Python由荷兰数学和计算机科学研究学会的吉多·范罗苏姆于1990年代初设计，作为一门叫做ABC语言的替代品。**Python**提供了高效的高级数据结构，还能简单有效地面向对象编程。**Python**语法和动态类型，以及解释型语言的本质，使它成为多数平台上写脚本和快速开发应用的编程语言。
